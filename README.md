# squidbot

<img width="724" height="636" alt="squidbot" src="https://github.com/user-attachments/assets/d2bd0975-0a21-4a6b-8ff6-03350c4f7078" />


squidBot is a versatile cybersecurity tool designed for social engineering and penetration testing operations. It empowers security professionals to simulate real-world attack scenarios, assess human vulnerabilities, and strengthen organizational defenses. Through seamless integration with platforms like Telegram, Discord, WhatsApp, and Slack, SquidBot enables users to issue commands remotely and automate engagement workflows. The tool supports customizable scripts, data collection, and reporting features for efficient analysis. Built with flexibility and control in mind, SquidBot enhances red team activities while maintaining operational efficiency. It is intended strictly for ethical use by authorized personnel conducting security assessments and awareness training in approved environments.

# How to clone the repo
```bash
git clone https://github.com/Iankulani/squidbot.git
cd squidbot
```

 # How to run
 ```bash
python squidbot.py
```
